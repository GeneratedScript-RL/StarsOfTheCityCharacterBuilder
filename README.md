# Stars of the City Character Sheet Builder

To be used for character building for stars of the city characters.

## Features

- Character identity, archetype, story ability, battle ability, and tragic flaw fields
- Level-aware totals for feats, known skills, improvements, Base E.G.O, and spare modules
- Stat controls for Might, Vitality, Agility, Intellect, Instinct, and Persona
- Derived HP, Stagger Resist, Speed Die, Max Light, Emotion Points, and Excellence
- Freeform decimal affinity values for every damage and stagger type
- Skill base picker, spare module allocation, notes, import/export JSON, print view, and local autosave

## Tech Stack

- React
- TypeScript / TSX
- Vite
- Vercel

## Local Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run lint
npm run build
```

The production output is written to `dist/`.
